## Commands and Queries

### 1) Book
+ CreateBook
+ UpdateBook
+ DeleteBook
+ GetBookDetail(GetById)
+ GetBooks(GetAll)

### 2) Author
+ CreateAuthor
+ UpdateAuthor
+ DeleteAuthor

+ TDD(Test Driven Development)
